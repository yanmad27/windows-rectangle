window-rectangle

Snap any window on Windows with a hotkey. Inspired by macOS Rectangle.

Download for Windows View on GitHub

~1–2 MB exe · Windows 10 / 11 · Free & open source (MIT)

What it does

  • • Halve the active window left, right, top, or bottom.
  • • Drop it into any corner as a quarter.
  • • Maximize toggle, center, restore previous size.
  • • Press the same halve hotkey again to throw the window onto the adjacent monitor.
  • • Tray menu lists every binding; clicking acts on the previously focused window.
  • • Per-monitor DPI aware. Single exe. No installer.

Default hotkeys

Hotkey Action
Ctrl+Alt+Left Halve Left (repeat → previous monitor)
Ctrl+Alt+Right Halve Right (repeat → next monitor)
Ctrl+Alt+Up Halve Top
Ctrl+Alt+Down Halve Bottom
Ctrl+Alt+U / I / J / K Quarter TL / TR / BL / BR
Ctrl+Alt+Enter Maximize toggle
Ctrl+Alt+C Center
Ctrl+Alt+R Restore previous size

Install

  1. Download the latest window-rectangle-*.exe from the releases page.
  2. Double-click to run. A blue-and-green window icon appears in the system tray.
  3. Hotkeys are active immediately. Right-click the tray icon for the menu.
  4. To launch on login, drop a shortcut into shell:startup.
  5. To quit: tray icon → Quit.

Build from source

Requires Rust stable and the x86_64-pc-windows-msvc target. 

cargo build --release --target x86_64-pc-windows-msvc

Output: target\x86_64-pc-windows-msvc\release\window-rectangle.exe.
